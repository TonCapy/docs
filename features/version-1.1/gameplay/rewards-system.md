# Rewards System

Rewards are distributed across all events with the following structure:

**#1**: 10% of prize pool

**#2**: 8%

**#3**: 5%

**#4-10**: 21% (3% each)

**#11-20**: 20% (2% each)

**#21-50**: 36% (1.2% each)

After 24 hours, users can view event history and claim rewards instantly, which will be credited to their balance.
